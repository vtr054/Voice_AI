Setup Guide

1. Initial Configuration

Go to Settings and configure LiveKit, Google Gemini, and Vobiz SIP credentials.

Once saved, click the ⚡ Trunk button under Vobiz to create the SIP trunk in your LiveKit project.

2. Global AI Prompt

Go to AI Prompt and customise the instructions. The AI uses this unless overridden by an Agent Profile.

3. Agent Profiles

Go to Agents and create personas with different voices and models.

4. Test Call

Go to Single Call, enter your phone number in E.164 format, and dispatch a call to test.
